# 🛒 MarketSoft

Aplicación web fullstack para la gestión de usuarios y productos
---

## 🚀 Tecnologías utilizadas

### 🔙 Backend
- Node.js
- Express
- PostgreSQL
- Sequelize
- Swagger (documentación API)

### 🎨 Frontend
- React
- Axios
- React Router DOM
- Bootstrap

---

## 📌 Funcionalidades

### 👤 Usuarios
- ✅ Crear usuario
- ✅ Listar usuarios
- ✅ Editar usuario
- ✅ Eliminar usuario

### 📦 Productos
- ✅ Crear producto
- ✅ Listar productos
- ✅ Editar producto
- ✅ Eliminar producto
